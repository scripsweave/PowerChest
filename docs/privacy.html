Privacy Policy

Last updated: 31 March 2026

The short version: PowerChest does not collect, transmit, or sell your data. The app runs entirely on your Mac. There is no analytics, no telemetry, no tracking, and no account system.

The App

PowerChest is a native macOS application that modifies system settings on your local machine. All operations happen locally. The app does not connect to any server, API, or cloud service during normal operation.

Specifically, PowerChest:

  • Does not collect usage analytics or telemetry
  • Does not transmit any data over the network
  • Does not require an account, login, or registration
  • Does not include any third-party SDKs, frameworks, or tracking libraries
  • Does not use crash reporting services

Data Stored Locally

PowerChest stores the following data on your Mac in ~/Library/Application Support/PowerChest/:

  • Snapshots — JSON files recording the state of your settings at a point in time. These are used for the Time Machine / restore feature.
  • Change log — a JSON file recording what settings were changed, when, and from which mode. This is your local audit trail.

This data never leaves your machine. You can delete it at any time by removing the ~/Library/Application Support/PowerChest/ directory.

Exported Profiles

When you use the Export Config feature, PowerChest saves a .powerchestprofile file to a location you choose. This file contains a snapshot of your settings. What you do with that file (share it, upload it, delete it) is entirely up to you. PowerChest does not upload it anywhere.

Admin Privileges

Some settings (firewall, SSH, captive portal, MAC address) require administrator privileges. When you apply these settings, macOS shows its standard password dialog. PowerChest does not store, cache, or have access to your admin password — authentication is handled entirely by macOS.

Apple Notarization

The distributed DMG is signed and notarized through Apple's notarization service. When you first open the app, macOS may check the notarization ticket with Apple. This is a standard macOS behavior and is not controlled by PowerChest.

This Website

This website is a static HTML page hosted on GitHub Pages (or your chosen hosting provider). It does not use any analytics, tracking pixels, or advertising networks.

However, like any website:

  • The hosting provider may collect standard server access logs (IP address, timestamp, page requested, browser user agent). This is a standard function of web servers and is not controlled by PowerChest.
  • The website does not set any cookies itself. Your browser may store standard caching headers.
  • GitHub Pages, if used as the hosting provider, has its own privacy statement.

Third-Party Links

This website links to GitHub (for source code and downloads). GitHub has its own privacy policy. PowerChest is not responsible for the privacy practices of third-party websites.

Children

PowerChest is a system utility for macOS. It is not directed at children under 13 and does not knowingly collect information from children.

Changes to This Policy

If this policy changes, the updated version will be posted here with a new "last updated" date. Since PowerChest collects no data, significant changes are unlikely.
